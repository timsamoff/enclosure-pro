Enclosure Pro is pretty straightforward to use and presents very few menu options that should be familiar to any effects pedal builder. Experiment for a few minutes and you'll get the hang of it.

1. Top Toolbar

The Top Toolbar provides access to information about the app, some helpful Design Canvas tools, and the Project Menu.

A. App Menu

The App Menu shows the app version number and developer contact info.

B. Zoom

Use the + and - buttons or the mouse scroll wheel to zoom in and out of the Design Canvas.

C. Rotate

The Rotate button can rotate the Design Canvas 90° clockwise and 90° counterclockwise.

D. Project Menu

The Project Menu defaults to "Untitled" and reflects the saved project name. If an orange asterisk (*) is next to the file name, it means that the most recent changes haven't been saved.

The Project Menu contains the following options:

  • New: Start a new enclosure project (Ctrl/Cmd+N)
  • Open: Open a saved enclosure project (Ctrl/Cmd+O)
  • Save: Save the open project as an .enc file (Ctrl/Cmd+S)
  • Save As: Save the open project as a new file (Ctrl/Cmd+Shift+S)
  • Print: Print the project blueprint (Ctrl/Cmd+P)
  • Export PDF: Export the project blueprint to PDF (Ctrl/Cmd+E)
  • Quit: Quit Enclosure Pro (Ctrl/Cmd+Q)

2. Bottom Toolbar

The Bottom Toolbar provides access to the enclosures, a Unit Switcher, Grid Settings, and the available components.

E. Enclosures Menu

This menu contains a list of the most common pedal enclosure sizes. Switching enclosures will not delete components already placed on the Design Canvas.

F. Unit Switcher

This switches between Metric (mm) and Imperial (in) units app-wide.

G. Grid Settings Menu

Provides several grid spacing options and the option to turn the grid off.

H. Components Menu

This provides a list of common enclosure components (potentiometers, switches, LEDs, etc.). Selected components appear in the center of the enclosure.

3. Design Canvas

The Design Canvas is where components are placed, supporting drag-and-drop functionality.

To delete a selected component, click the orange trash can icon that appears at the bottom right of the Design Canvas. Use the mouse right-click to pan around the Design Canvas space.

Printing & Drilling Guide

After printing the drill hole layout, follow these steps:

  1. Cut it out (a blade and a straight edge are recommended over scissors).
  2. Cut the enclosure template out with or without the rounded corners.
  3. Fold the template along the face (front) edges and wrap it around the enclosure.
  4. Tape the template in place.
  5. Use the crosshairs on the template to guide your center punching.
  6. Remove the template and drill!
