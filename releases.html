Version History

v1.1.0 (December 18, 2025) 
  • New Start screen for selecting a starting enclosure
  • Legacy enclosure updates for old version saves
  • Refined app auto-update
  • Enclosures now auto-fit to the screen space
  • Removed direct print in place of exporting to PDF (PDF exports print more precisely)
    • Includes a new “print instructions” screen
  • Increased export to PDF resolution
  • Export to PDF now auto-selects proper paper sizes for enclosures
  • Refined header and footer info in PDFs
  • Added calibration mark in PDFs for printing accuracy
  • Panning is now accomplished via middle-click
  • Right-clicking a selected component will display a context menu with Duplicate, Rotate 90°, and Print/Export to PDF (on/off)
  • Added enclosure manufacturer categories with specific enclosure sizes
    • Manufacturers listed in alphabetical order
    • Enclosure manufacturers also have brand-specific color coding and badges
  • Added component categories
  • Changed component menu icon
  • Added “Footprint Guides” for determining drill hole placement (internal and external footprints available)
  • Refined UX for file saving/opening and confirmation dialogs
  • Updated all cursor iconography app-wide
  • Added specific file icon for saved files
  • Updated documentation
  • Switched from PayPal to Ko-fi for donations
v1.0.1 (November 26, 2025) 
  • Fixed critical rotation bug
  • Fixed auto-update functionality
  • Added Feedback option to App Menu
v1.0.0 (November 24, 2025)
  • Initial Release: Cross-platform (Windows, Mac, Linux) application launch
  • Core functionality: Precision drill hole layout for standard effects pedal enclosures (1590B, 125B, 1590BB)
  • Drag-and-drop component library
  • Component delete functionality
  • Centered grid lines for accurate measurements
  • Multi-side design views (top, sides, bottom)
  • Metric and imperial unit support
  • Zoom and rotate
  • Grid settings/no grid
  • Project save/load
  • PDF template generation
  • Integrated print
